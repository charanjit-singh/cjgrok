# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.0] - 2024-01-XX

### Added
- Popup interface for accessing All Posts modal
- Path-based button visibility (only shows on /imagine page)
- Beautiful semi-transparent UI with blur effects
- GitHub Actions workflow for automated releases
- Semantic versioning support

### Changed
- Button text changed from "CJGrok" to "All Posts"
- Improved modal design with better blur and transparency effects
- Enhanced card hover effects and styling

## [1.1.0] - 2024-01-XX

### Added
- Header capture from browser requests
- Dynamic header usage for API calls
- Date formatting in local timezone

## [1.0.0] - 2024-01-XX

### Added
- Initial release
- Gallery view of all Grok Imagine posts
- Delete individual posts
- Bulk delete functionality
- Thumbnail preview with prompts
